# Frame Shop — Project & Billing Tracker

A management interface that lets a custom framing shop track billing, materials and tools used on each job, and which team members are assigned to which projects — all in one place.

## What it does

- **Billing tracking** — record charges per project and keep a running view of what's owed and what's been paid.
- **Tools & materials** — log the tools, moulding, glass, and other materials used on each job so costs are tied to the work.
- **Team assignment** — see who is working on which project inside the shop, so nothing falls through the cracks.

## Why I built it

Built for a real custom framing business that was tracking jobs by hand. The goal was to replace scattered notes and spreadsheets with a single, easy-to-read interface the shop owner and staff could actually use day to day.
